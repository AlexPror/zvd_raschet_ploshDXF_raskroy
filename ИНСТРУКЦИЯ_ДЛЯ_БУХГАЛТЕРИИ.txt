  ИНСТРУКЦИЯ ПО УСТАНОВКЕ И ИСПОЛЬЗОВАНИЮ
  ZVD Nesting Calculator
═══════════════════════════════════════════════════════════════

📦 УСТАНОВКА ПРОГРАММЫ
───────────────────────────────────────────────────────────────

1. Дважды кликните на файл:
   "ZVD Nesting Calculator Setup 1.0.0.exe"

2. Следуйте инструкциям установщика:
   - Нажмите "Далее" (Next)
   - Выберите папку установки (можно оставить по умолчанию)
   - Нажмите "Установить" (Install)
   - Дождитесь завершения установки
   - Нажмите "Готово" (Finish)

3. Программа будет установлена и создаст ярлык на рабочем столе.

⚠️ ВАЖНО: Если программа не запускается, возможно нужно установить Python.
   См. раздел "Установка Python" ниже.

───────────────────────────────────────────────────────────────

🚀 ЗАПУСК ПРОГРАММЫ
───────────────────────────────────────────────────────────────

1. Найдите ярлык "ZVD Nesting Calculator" на рабочем столе
   ИЛИ
   Найдите программу в меню "Пуск" → "ZVD Nesting Calculator"

2. Дважды кликните на ярлык

3. Программа откроется в отдельном окне

───────────────────────────────────────────────────────────────

📋 ИСПОЛЬЗОВАНИЕ ПРОГРАММЫ
───────────────────────────────────────────────────────────────

1. ЗАГРУЗКА ФАЙЛОВ:
   - Перетащите DXF файлы в область загрузки
   ИЛИ
   - Нажмите "Выбрать файлы" и выберите DXF файлы

2. ПРОВЕРКА ДАННЫХ:
   - Проверьте количество деталей (автоматически определяется)
   - При необходимости измените количество вручную

3. ВЫБОР МАТЕРИАЛА:
   - Выберите материал из списка
   - Укажите цену за м² (если нужно)

4. РАСЧЕТ РАСКРОЯ:
   - Нажмите кнопку "Рассчитать раскрой"
   - Дождитесь завершения расчета
   - Просмотрите визуализацию размещения деталей

5. ЭКСПОРТ РЕЗУЛЬТАТОВ:
   - Нажмите "Экспорт в Excel" для создания таблицы
   - Нажмите "Экспорт в PDF" для создания отчета
   - Файлы сохранятся в папку "Загрузки"

───────────────────────────────────────────────────────────────

🐍 УСТАНОВКА PYTHON (если программа не запускается)
───────────────────────────────────────────────────────────────

Если при запуске появляется ошибка о Python:

1. Скачайте Python с официального сайта:
   https://www.python.org/downloads/

2. При установке ОБЯЗАТЕЛЬНО отметьте:
   ☑ "Add Python to PATH" (Добавить Python в PATH)

3. Установите Python

4. Перезапустите программу

───────────────────────────────────────────────────────────────

❓ РЕШЕНИЕ ПРОБЛЕМ
───────────────────────────────────────────────────────────────

ПРОБЛЕМА: Программа не запускается
РЕШЕНИЕ: 
  - Установите Python (см. раздел выше)
  - Перезагрузите компьютер
  - Попробуйте запустить от имени администратора

ПРОБЛЕМА: Ошибка при загрузке файлов
РЕШЕНИЕ:
  - Убедитесь, что файлы имеют расширение .dxf
  - Проверьте, что файлы не повреждены
  - Попробуйте загрузить файлы по одному

ПРОБЛЕМА: Не работает экспорт
РЕШЕНИЕ:
  - Проверьте права доступа к папке "Загрузки"
  - Убедитесь, что есть свободное место на диске
  - Попробуйте сохранить в другую папку

───────────────────────────────────────────────────────────────

📞 ТЕХНИЧЕСКАЯ ПОДДЕРЖКА
───────────────────────────────────────────────────────────────

При возникновении проблем:
1. Проверьте эту инструкцию
2. Сделайте скриншот ошибки
3. Обратитесь к разработчику

───────────────────────────────────────────────────────────────

✅ ТРЕБОВАНИЯ К СИСТЕМЕ
───────────────────────────────────────────────────────────────

- Windows 10 или новее
- Свободное место на диске: 500 МБ
- Python 3.8 или новее (если не встроен в установщик)
