# Changelog

Все значимые изменения в проекте будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.0] - 2025-01-XX

### Добавлено
- Загрузка и парсинг DXF файлов
- Расчет площадей деталей с учетом зазоров
- Оптимизация раскроя на листах материала (2500x1250 мм)
- Визуализация размещения деталей на канвасе
- Валидация раскроя (проверка пересечений и выходов за границы)
- Экспорт результатов в Excel с полными таблицами
- Экспорт результатов в PDF с визуализацией
- Поддержка поворота деталей для оптимизации
- Расчет стоимости с учетом цены материала
- Автоматическое извлечение количества деталей из имени файла
- Автоматическое извлечение номера заказа из имени файла
- Поддержка различных материалов (оцинковка, нержавейка)
- Печать страницы в PDF через браузер

### Исправлено
- Проблемы с кодировкой кириллицы в PDF
- Ошибки с объединенными ячейками в Excel
- Проблемы с переименованием файлов при экспорте
- Корректное вычисление зазоров между деталями
- Обработка поворота деталей при размещении

### Изменено
- Улучшена структура экспорта Excel (добавлены заголовки таблиц)
- Оптимизированы стили печати для PDF
- Улучшена визуализация раскроя
